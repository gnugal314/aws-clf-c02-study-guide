# Scenario-Based Question Bank

## Cloud Concepts

1. A startup wants to avoid purchasing servers before understanding demand. What AWS benefit applies?
   - Answer: Trade fixed expense for variable expense.
2. A retailer has unpredictable holiday traffic. What capability should it use?
   - Answer: Elasticity.
3. A company wants faster infrastructure provisioning. What cloud benefit applies?
   - Answer: Agility.
4. A business wants to stop maintaining datacenters. What benefit applies?
   - Answer: Reduced infrastructure management.
5. A company wants to expand globally quickly. What AWS benefit applies?
   - Answer: Go global in minutes.
6. A company wants lower cost through shared provider scale. What concept applies?
   - Answer: Economies of scale.
7. A team wants to run experiments without long-term infrastructure commitment. What concept applies?
   - Answer: Pay-as-you-go.
8. A company wants cloud plus on-premises resources. What model fits?
   - Answer: Hybrid cloud.
9. A company wants dedicated internal resources. What model fits?
   - Answer: Private cloud.
10. A company wants fully cloud-hosted applications. What model fits?
   - Answer: Cloud deployment.

## Compute

1. A company needs full OS control. Which service should it use?
   - Answer: EC2.
2. A team wants to run code without managing servers. Which service fits?
   - Answer: Lambda.
3. A company uses Kubernetes and wants AWS-managed control plane. Which service fits?
   - Answer: EKS.
4. A team wants containers without managing EC2 instances. Which service fits?
   - Answer: Fargate.
5. A company wants AWS-native container orchestration. Which service fits?
   - Answer: ECS.
6. A batch workload can run later and scale massively. Which service fits?
   - Answer: AWS Batch.
7. A simple website needs easy cloud hosting. Which service fits?
   - Answer: Lightsail.
8. A web app needs managed deployment with reduced infrastructure work. Which service fits?
   - Answer: Elastic Beanstalk.
9. A fault-tolerant workload needs lowest compute cost. Which option fits?
   - Answer: Spot Instances.
10. A variable workload needs automatic capacity changes. Which service fits?
   - Answer: Auto Scaling.

## Storage

1. A company needs durable storage for images and backups. Which service fits?
   - Answer: S3.
2. An EC2 instance needs persistent disk storage. Which service fits?
   - Answer: EBS.
3. Multiple Linux EC2 instances need shared files. Which service fits?
   - Answer: EFS.
4. A company needs long-term archive storage. Which service fits?
   - Answer: S3 Glacier.
5. A company wants centralized backup. Which service fits?
   - Answer: AWS Backup.
6. A company wants hybrid on-premises and AWS storage. Which service fits?
   - Answer: Storage Gateway.
7. A Windows workload needs managed file storage. Which service fits?
   - Answer: FSx for Windows File Server.
8. A high-performance computing workload needs fast file storage. Which service fits?
   - Answer: FSx for Lustre.
9. A company wants to move older objects to cheaper tiers automatically. Which feature fits?
   - Answer: S3 lifecycle policy.
10. A static website needs low-cost hosting. Which service fits?
   - Answer: S3 static website hosting.
