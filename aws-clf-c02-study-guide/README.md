# AWS CLF-C02 Cloud Practitioner Study Guide

A GitHub-ready 14-day study packet for the **AWS Certified Cloud Practitioner CLF-C02** exam.

This repo is designed for busy working professionals who need a structured, practical, scenario-based way to prepare for the exam.

## What This Repo Includes

- 14-day study plan
- Daily study topics
- Daily practice questions
- Scenario-based question bank
- AWS service-to-use-case mapping
- Networking deep dive
- Exam keyword trigger cheat sheet
- Service weight study sheet
- Copy/paste study templates
- Simple architecture diagrams

## Recommended Study Time

For a working professional with limited time:

- Minimum effective plan: 90 minutes/day
- Recommended plan: 2 to 2.5 hours/day
- Weekend acceleration: 3 to 4 hours/day if available

## How to Study

1. Read the daily study guide.
2. Review the service keyword triggers.
3. Complete daily practice questions.
4. Complete scenario-based questions.
5. Use the service mapping sheet to connect business problems to AWS services.
6. Review weak areas before moving to the next day.

## Core Study Principle

Do not memorize AWS services in isolation.

Study each service as a business problem solver:

- What problem does it solve?
- What operational benefit does it provide?
- Is it managed or customer-managed?
- Is it serverless?
- What trigger words point to it on the exam?
