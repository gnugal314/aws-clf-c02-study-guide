# AWS Service to Use Case Map

## Compute

| Use Case | AWS Service |
|---|---|
| Full control of virtual servers | EC2 |
| Event-driven code without servers | Lambda |
| Container orchestration | ECS |
| Managed Kubernetes | EKS |
| Containers without server management | Fargate |
| Simple application deployment | Elastic Beanstalk |
| Batch processing | AWS Batch |

## Storage

| Use Case | AWS Service |
|---|---|
| Scalable object storage | S3 |
| Persistent EC2 disk | EBS |
| Shared Linux filesystem | EFS |
| Long-term archive | S3 Glacier |
| Centralized backup | AWS Backup |
| Hybrid cloud storage | Storage Gateway |

## Database

| Use Case | AWS Service |
|---|---|
| Managed SQL database | RDS |
| High-performance relational DB | Aurora |
| NoSQL key-value database | DynamoDB |
| Data warehouse | Redshift |
| In-memory cache | ElastiCache |
| Graph relationships | Neptune |
| MongoDB-compatible database | DocumentDB |

## Networking

| Use Case | AWS Service |
|---|---|
| Private network | VPC |
| DNS routing | Route 53 |
| Global CDN | CloudFront |
| Load balancing | ELB |
| Dedicated private connectivity | Direct Connect |
| Encrypted VPN tunnel | Site-to-Site VPN |
| Central network hub | Transit Gateway |

## Security

| Use Case | AWS Service |
|---|---|

| Encryption keys | KMS |
| DDoS protection | Shield |
| Web traffic filtering | WAF |
| Threat detection | GuardDuty |
| Sensitive data discovery | Macie |
| Compliance reports | Artifact |
| Secrets storage and rotation | Secrets Manager |

| Use Case | AWS Service |

| Use Case | AWS Service |
|---|---|
