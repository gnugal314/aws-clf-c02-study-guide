# Study Strategy for Busy Professionals

## Recommended Daily Commitment

For a working professional, parent, spouse, and caretaker, the best balance is:

## 2 to 2.5 focused hours per day for 14 days

This is enough to build recognition, reinforce concepts, and practice exam-style scenarios without creating burnout.

## Minimum Effective Plan

If time is extremely limited:

| Activity | Time |
|---|---:|
| Learn core concept | 45 minutes |
| Practice questions | 30 minutes |
| Flashcard review | 15 minutes |

Total: approximately 90 minutes/day

## Recommended Daily Structure

| Block | Time | Focus |
|---|---:|---|
| Learn | 45–60 minutes | Read/watch core concept material |
| Active Recall | 30 minutes | Flashcards, keyword review, verbal recall |
| Scenario Practice | 45–60 minutes | Service-to-use-case matching |
| Quick Review | 15 minutes | Weak areas and cram sheet |

## Highest ROI Study Method

Focus less on memorizing service definitions and more on matching AWS services to business scenarios.

Example:

| Business Need | AWS Service |
|---|---|
| Run code without managing servers | AWS Lambda |
| Store objects, logs, backups, images | Amazon S3 |
| Managed relational database | Amazon RDS |
| NoSQL key-value database | Amazon DynamoDB |
| DNS routing | Amazon Route 53 |
| Monitor metrics and logs | Amazon CloudWatch |
| Audit AWS API calls | AWS CloudTrail |
